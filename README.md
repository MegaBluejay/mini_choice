# mini_choice

## Dependencies
python3, modules:

```
toolz, tabulate, termcolor
```

## Installation

recommended method of installation is from pypi

```
pip install --user --index-url https://test.pypi.org/simple/ mini_choice
```

## Usage

```
choice [-h] [dir]
  -h, --help  show this help
  dir         scenes folder. default is ./scenes
```
